A lightweight, purely root-based Quick Settings tile to switch the default mobile data SIM on dual-SIM Android devices.

Unlike other implementations that rely on Shizuku and persistent background daemons, Data SIM Switcher uses topjohnwu's libsu to spawn a Root IPC service for the exact millisecond the tile is tapped. It is incredibly fast, survives reboots reliably, and uses zero background battery.

Features:
• Zero Daemons: Uses on-demand Magisk/KernelSU root access.
• No Permissions Required: Bypasses Android's READ_PHONE_STATE restrictions entirely by doing all calculation logic natively in the root daemon.
• Dynamic Tile Subtitle: Displays the name of the currently active data SIM (Requires Android 10+).
• Smart Toggling: Automatically scans active slots, calculates the current default data SIM, cycles to the next one, and securely enables the cellular radio on the new SIM.

Requires Root.
